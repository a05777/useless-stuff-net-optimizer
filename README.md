# 一个没啥用的网络优化器

![1](https://img.shields.io/badge/License-GPLv3-blue.svg)

因为我有一朋友非常信任网络优化器，说自从用了网络优化器，打人不空刀了，连击也流畅了，但是200块钱对于我来说我显然是用不起的，并且我不信这玩意，所以我打算自己（AI）写一个
***
主要原理：

改MTU，防止MTU分片导致的延迟激增或丢包

改QoS策略，把目标域名解析出来的IP给予高优先级，让其在本地网关和系统内有更高的优先级，但是对于运营商骨干网没用，***世界加钱可及***

关闭Nagle算法，这个算法是Windows为了省带宽做出来的，把小包攒成一堆再发出去，导致延迟增加，但是现代游戏基本都关闭了Nagle，基本无用

关闭网卡的中断节流，EEE（节能以太网），LSO（大发包载卸）
